>**三角平方数定理**：
>1. 方程$$x^2 - 2y^2 = 1$$的每个正整数解都可以通过将$3 + 2\sqrt{2}$自乘得到，即解$(x_k, y_k)$可以通过展开下式得到。$$x_k + y_k\sqrt{2} = (3 + 2\sqrt{2})^k，k = 1，2，3，···$$
>2. 每个三角平方数$n^2 = {1 \over 2}m(m + 1)$由$$m = {{x_k - 1} \over 2}，n = {y_k \over 2}，k = 1，2，3，···$$给出，其中$(x_k, y_k)$是由1得到的解
